# Portfolio - Barbara Drummond

Personal portfolio website built to showcase professional background, featured projects, technical skills, and contact channels.

## Live site

- URL: https://babidrummond.github.io/portfolio/
- Hosting: GitHub Pages

## Overview

This project is a single-page website focused on:

- Professional presentation
- Project and experience showcase
- Smooth navigation between sections
- Responsive design for desktop and mobile

## Technologies used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap 5
- AOS (Animate On Scroll)
- Typed.js

## Project structure

```text
.
├── index.html
└── assets
    ├── css
    │   └── style.css
    ├── img
    └── js
        └── main.js
```

## Page sections

- Home
- About
- Portfolio
- Resume
- Skills
- Contact

## Interface features

- Fixed top navigation bar
- Scroll reveal animations with AOS
- Animated typing text with Typed.js
- Responsive layout
- Basic loading optimizations (preconnect, dns-prefetch, and deferred loading for some resources)
